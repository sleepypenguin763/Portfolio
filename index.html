Details About This Project


This was the fist time for me to develop a React application from scratch. I decided to use OpenSky API to retreive real-time flight data, and decided to present that on my webpage. At any point in time, there are at least 4500 flights being operated, so fetching this many data point caused efficiency problems.

OpenSky API only provides the data regarding a flight's altitude, speed, and location, so I made my own mock REST API using publically available dataset that maps the callsign of the plane to its route (origin and destination). This caused my application to make thousands of API requests when loading the page, leading to slower page loading time. Since the assignment required to only use one page, I was not able to implement pagination/multi-page application, but in order to overcome Google Chrome's error of overly making API request at one time (and to increase load efficiency), I decided to load the data in batch.

For more details, please view my github repository.
