Persona & Story Board

Contextual Inquiry: Observation of PlayStation 4 Controller

Today, there are various gaming consoles and devices that people play games with. I have decided to focus on one of them, a Play Station 4 controller. Playstation is one of the most widely used gaming consoles and the controller has a lot of buttons and keys that the user can interact with. Furthermore, this interface is in public view and it does not only have simple interactive components. The controller contains multiple buttons, joysticks, triggers, and a touch pad.


User Interface Questions: Use and Experience of PS4 Controller Interface

Below are the questions I came up with to ask about the experiences using the interface. It mainly asks about what to use the console for, since that heavily affects the use of the controller.

    What kind of features do you use in Playstation?
  • Do you use it for gaming?
  • Do you use the controller when using the keyboard function?
  • Do you ever go to the Play Station Store using the controller? If so, how do you feel about navigating through using the controller there?
  • Do you prefer other modes of controlling such as mouse / keyboard rather than using the keyboard?

    How are the experiences of using the controller?
  • Does certain key stop functioning often? If so, which key is most common?
  • How often do you replace your controller?
  • If any, is there any buttons/sticks that should be added/removed from the current controller?
  • Do you prefer wired or wireless controllers?
  • How often do you change the key mapping of the controller?
  • Are all keys reachable when holding the controller using two hands?
  • Have you ever ended up with sore hand after holding on to the controller?

Interview and Observation: Comfortableness of PS4 Controller

I observed three users using the play station. Observation section contains the objective Observation of each user's use case. Interview section is what the user responded to the given questions. These questions are also noted in the User Interface Questions section of this page.

>

Observations

    User 1
  • Used the original wireless controller sold by PlayStation
  • All inputs were done using the controller
  • Played first person shooting (FPS) game
  • All log-in actions were taken using the controller
  • Played the game. Since this is a FPS game, this person used more joy sticks and left right trigger than other keys
  • Did not touch the in-game key mapping settings
  • Played over one hour, did not make comments about the hand being sore from holding onto the controller for long period of time

    User 2
  • Used wired controller that they bought separately (Still an official controller sold by Sony)
  • Also used the converter that allows the use of keyboard and mouse
  • All alphabetical inputs were done using the keyboard instead of the controller
  • Played open world game with some fighting / actions involved
  • Changed some key mapping on the controller through the in game setting
  • Went back to controller when entering the game
  • Thumb sticks / joy sticks are very often used to travel around
  • Played for around an hour, no comments about the controller fit

    User 3
  • Used wired controller
  • Walked into the store to browse through potential game to purchase
  • Watched promotion video
  • Purchased the game and used controller throughout(asked after completing the payment)
  • Mainly used the arrow keys on the left hand side of the controller to navigate through the store
  • Not much use of the controller compared to previous two users as this person did not end up playing the game for the day
  • Was about to trip over the wire portion of the controller when walking around

Interview

    What kind of features do you use in Playstation?
  • Controllers are mainly used for gaming and browsing through settings / stores
  • None of the user I interviewed streamed a game through PS4 so no opinions on that
  • Users never used web browser within PlayStation

    How often do you replace your controller? What is the main reason for replacement?
  • User 1 mentioned that the joy stick sometimes stops functioning. This user assumed that it is because FPS games require consistent joystick movement, causing the sticks to deplete quickly.
  • User 2 and User 3 never had a need of replacing the controller nor the cable
  • Major concerns about the controller was that joy sticks and triggers are easier to break than the buttons

    If any, is there any functionality that should be added / removed / changed from the current controller?
  • Input that uses Keyboard on the monitor screen can be annoying to use from time to time
  • Setting up wired and wireless mics and headphones are sometimes confusing
  • Direction buttons on the left hand side of the controller is a bit small
  • Pressing down on the joy stick to use it as a button can be hard without moving the joy stick

    Do you prefer wired or wireless controllers?
  • All users prefered wireles controller over wired ones
  • Player does not have to worry about the length of the cable when using the wireless controller
  • Easier access and use of mouse and keyboard (aka without the use of converters), especially when not playing games can be useful

    Does the controller fit well to your hand?
  • Joy sticks are a bit too large for people with smaller hands
  • Sometimes encounter the difficulty of pressing the X button while using the right joystick
  • All users said that the controller fits their hand well in general, and does not hurt even after holding on to it for long period of tiem

Persona

Below are the two Persona I came up with according to the interview results from the previous section. The template for the below empathy map are taken from Figma.

Interface Problems
    Persona 1
  • Using the UI keyboard with a controller is annoying and hard. It takes far more time to type things using the controller than when typing on a computer
  • Sometimes, it takes some time for the key input to be reflected on the console side
  • With wired controllers, the wire can get in the player's way

    Perona 2
  • The user is having trouble figuring out the correct sensitivity of joysticks
  • Certain buttons, especially the square button is hard to press while using other buttons
  • Plugging in the wired headphone / mic on the controller causes the wire to interrupt key inputs. The mic plug in location should be improved
  • Default key mapping differs between countries / continent
  • Pressing down the joystick vertically is hard without tilting or moving it.
How and Why each Persona Represents the Users

    Both personas have some similarities with some differences. First, both personas represent the user who does not use PS4 from day-to-day basis, and also focuses on the fact that the user has some experiences using a computer and/or other consoles such as XBox. Now, the first thing that is most important to note is that the PS4 controller must be used with the PS4 in this case. It is true that there are softwares that allows PS4/XBox controllers to be used on computers, but in both personas, I assumed that PS4 controllers are only used when using PS4.

    In the case of persona 1, I focused on how the user would feel and experience when touching the PS4 / PS4 controller for the first time. Clearly, one has to set up their account, purchase a game, connect to the internet, etc. Most or all of these steps require a user to use the controller, and most importantly, it involves a lot of console side UI keyboard inputs (ex. typing keyword when searching for an app, entering wifi password). This can take a lot of time especially when compared to keyboard inputs on a standard PC, so that is what I focused on. In other words, this persona focuses on potential additional features that could be added to the controller to ease the keyboard inputs.

    In the case of persona 2, I focused on the strengths and weaknesses of the controller itself. Figuring out sensitivity of the keys and buttons, how to plug microphones into the controller, how it fits the user's hand is focused. This persona is made from the previous interview section where users mentioned how the controller can not fit well for people with smaller hands, or if one has a big hand, that it is hard to press a certain combination of buttons at the same time.



Story Board

Below is the storyboard for persona 1 from previous section. The person is new to Play Station, and the end goal is to purchace and launch the game. However, since he is renting his friend's PlayStation, the original set up of the device is completed. He just needs to get used to the PS4 controller and get a sense of how to navigate through the platform. Although this story board might not seem to reflect the user experience of the PlayStation4 Controller, navigating through the console requires getting used to the controller. Thus, setting up wi-fi, going to the store, launching the game and taking a look at the key mapping setting does describe the user experience of using the controller.

    3. He tries to connect to the internet so that he can access the store

    Sam: "How do I use this keybaord?"

    Sam: "Arrow Buttons are useful here"

    Sam: "This takes so much time to input."

    6. After the game got installed, he opened the new game and looked throught the key mapping, testing out the new game.
